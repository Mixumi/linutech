Linutech 产品隐私政策

我们致力于构建安全、透明的数字工具。无论您使用 TimeHabit、GoalHabit、TimingHabit、TaskMatrix、LimStar 或 TastePost，所有产品均遵循相同的核心原则：尽量减少数据收集、本地优先存储、绝不出售个人信息，并提供清晰的控制选项。

TimeHabit

iOS · 专注与屏幕时间管理

我们收集的数据

  • TimeHabit 不会上传您选择的应用、使用时长或任何屏幕时间统计。
  • 设备分组、自动化规则和限制设置均仅保存在您的本地设备中。

使用的权限与存储

  • 需要 Screen Time API 授权以隐藏或限制应用，此授权由系统弹窗统一管理。
  • 可选开启 iCloud 同步，便于在多台设备间共享设置；数据加密后传输，Linutech 无法访问。

第三方服务

TimeHabit 不集成任何第三方分析、广告或追踪 SDK。

您的控制权

  • 可随时在 iOS 设置中撤销 Screen Time 权限。
  • 删除应用即自动清除本地和 iCloud（如启用）的 TimeHabit 数据。

GoalHabit

iOS · 目标与习惯追踪

我们收集的数据

  • GoalHabit 不会收集或上传任何目标、习惯或备忘数据。
  • 所有记录、提醒和完成状态都保存在用户设备的加密数据库中。

同步与备份

  • 可选启用 iCloud 同步以备份目标数据；Linutech 不会访问同步内容。
  • 关闭同步后，所有数据将仅保留在本机，随时可以导出备份。

第三方服务

GoalHabit 不集成广告或统计 SDK，不与第三方共享任何个人信息。

您的控制权

  • 可在应用内导出或删除所有记录。
  • 卸载应用时，所有本地数据将一并删除。

TimingHabit

iOS · 番茄钟与效率计时

我们收集的数据

  • TimingHabit 不会收集任何计时任务、标签或备注信息。
  • 历史计时记录只在本地生成，供您查看完成情况。

设备权限

  • 如启用提醒或通知，仅使用系统通知权限，不访问您的通讯录或定位信息。
  • 音效、提示音均来自系统资源，不会访问麦克风。

第三方服务

应用不接入任何第三方追踪、广告或崩溃统计服务。

您的控制权

  • 可手动清空历史记录并撤销通知权限。
  • 删除应用将清除全部本地数据。

TaskMatrix

Web & 桌面 · 项目协同

我们收集的数据

  • 注册账号仅需邮箱，用于创建团队空间和接收重要通知。
  • 任务、评论、文件等内容存储在受控的云端数据库，用于提供实时协同能力。

数据存储与安全

  • 所有传输均通过 HTTPS 加密。
  • 团队数据与系统日志隔离存储，限制内部访问权限并记录审计日志。

第三方服务

  • 使用行业标准的对象存储服务托管附件，仅用于文件交付。
  • 可选启用企业级单点登录（SSO），我们不会保存您的第三方登录凭证。

您的控制权

  • 管理员可导出或删除整个团队空间的数据。
  • 账号注销后，相关个人数据将在 30 天内从生产环境移除。

LimStar

iOS & Web · 灵感收集与创作

我们收集的数据

  • 保存的灵感、标签和附件文件默认存储在本地设备。
  • 如启用跨设备同步，将在云端创建加密副本，Linutech 不会用于广告或推荐。

访问的系统权限

  • 仅在您授权时访问相册或文件，以便导入图片或文档。
  • 可选使用相机拍摄灵感素材，拍摄内容不会离开设备除非您主动上传。

第三方服务

LimStar 不植入广告，不与第三方共享您的创作内容。

您的控制权

  • 可随时导出灵感库或彻底删除某个项目。
  • 关闭同步功能后，云端备份会自动清除。

TastePost

移动端 · 美食日记

我们收集的数据

  • TastePost 不会自动收集位置信息或联系人资料。
  • 餐厅记录、评分、照片由您自行创建，可选择是否同步到云端。

照片与定位权限

  • 仅在您主动添加照片时请求相册或相机权限，系统可随时撤销。
  • 可选手动填写地址或使用定位以便回顾，我们不会在后台持续追踪。

第三方服务

我们不使用广告联盟。若启用地图定位，仅调用系统地图服务，不会上传历史轨迹。

您的控制权

  • 应用内提供批量删除与数据导出工具。
  • 注销账号后，TastePost 会在 30 天内清除云端备份。
